Map marker states:
Ink / dark = Fully verified · Amber = Partially reported · Sage = Currently selected (active). Active marker scales 1.15x.

Variants:

03 — Components · Skeleton Loaders

Loading gracefully

Skeleton uses shimmer animation (1.4s, ease-in-out). Show skeletons immediately on page load — never show a spinner for list/grid views. Replace with real content as data arrives, staggering cards by 50ms each.
